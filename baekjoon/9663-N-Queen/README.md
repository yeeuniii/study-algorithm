# 💡**문제 분석 요약**

| 시간 제한 | 메모리 제한 |
| --- | --- |
| 10초 | 128MB |

N x N 체스판 위에 퀸 N개를 서로 공격할 수 없게 놓는 문제
<br>
즉, N이 주어졌을 때, 퀸을 놓는 방법의 수를 구하는 프로그램 작성
<br>

> 현재 퀸의 위치 (x, y)일 때 이동범위 (Suppose  -N < a < N)
> 1. 좌우. i.e. (x + a, y)
> 2. 위아래. i.e. (x, y + a)
> 2. 3. 대각선. i.e. (x + a, y + a) 

---
### 입력

첫째 줄 : N

- 1 ≤ n < 15

### 출력

N개를 서로 공격할 수 없게 놓는 경우의 수 출력

---
#### **예제**
- 입력
    
    ```
    8
    ```
    
- 출력
    
    ```
    92
    ```
