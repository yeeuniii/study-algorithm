# **문제 분석 요약**

| 시간 제한 | 메모리 제한 |
| --- | --- |
| 2초 | 256 MB |

1부터 N번까지 N명의 사람들이 원을 이루어 앉아 있을 때,

모든 사람이 없어질때까지 N보다 작은 임의의 정수번째 사람을 반복해서 제거

이때, 원에서 사람들이 제거되는 순서를 (N, K) 요세푸스 순열이라고 함.


🔖**예시** : (7, 3)-요세푸스 순열은 <3, 6, 2, 7, 5, 1, 4>



### 입력

첫번째 줄에 N과 K가 공백 구분으로 순서대로 주어짐 (1 ≤ K ≤ N ≤ 5,000)

### 출력

`<(N, K) 요세푸스 순열>`
