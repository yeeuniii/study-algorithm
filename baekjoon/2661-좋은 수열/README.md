# 💡**문제 분석 요약**

| 시간 제한 | 메모리 제한 |
| --- | --- |
| 1초 | 128MB |

숫자 1, 2, 3으로만 이루어지는 수열이 있음
<br>
이때, 인접한 길이의 두 개의 부분 수열이 동일한 것이 있으면 나쁜 수열. O.W. 좋은 수열

<aside>
🔖 나쁜 수열의 예

```
33
32121323
123123213
```

</aside>

<aside>
🔖 좋은 수열의 예

```
2
32
32123
1232123
```

</aside>

⇒ 길이가 N인 좋은 수열들을 N자리의 정수로 보아 그 중 작은 수열을 구하는 프로그램

---
### 입력

첫째줄 : 숫자 N

- 1 ≤ N ≤ 80

### 출력

길이가 N인 좋은 수열 중 가장 작은 수열


---
#### **예제**
- 입력
    
    ```
    7
    ```
    
- 출력
    
    ```
    1213121
    ```
