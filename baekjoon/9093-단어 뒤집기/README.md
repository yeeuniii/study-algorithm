# 문제 분석

| 시간 제한 | 메모리 제한 |
| --- | --- |
| 1초 | 128MB |

주어진 문장을 뒤집어서 출력하는 프로그램 

단, 단어의 순서는 바뀌지 않음

### 입력

첫째 줄 : 테스트 케이스 수 T

둘째 줄 ~ T개: 한 문장

- 단어의 길이 최대 20
- 문장의 길이 최대 1000

### 출력

입력으로 주어진 문장 뒤집기

- **예제**
    - 입력
        
        ```
        2
        I am happy today
        We want to win the first prize
        ```
        
    - 출력
        
        ```
        I ma yppah yadot
        eW tnaw ot niw eht tsrif ezirp
        ```
