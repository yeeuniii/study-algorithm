# 문제 분석

| 시간 제한 | 메모리 제한 |
| --- | --- |
| 1초 | 128MB |

    VPS인지 아닌지 판별하기
- PS(Parenthesis String, 괄호 문자열) : `(`와 `)` 로만 이루어진 문자열
- VPS(Valid PS, 올바른 괄호 문자열)


**🔖 예제**

| VPS | Not VPS |
| --- | --- |
| (())() | (()( |
| ((())) | (())())) |
|  | (() |


### 입력

첫째 줄: 입력 데이터 수. 정수 T

둘째 줄 ~ T개: 괄호문자열

- 문자열의 길이는 2 이상 50 이하

### 출력

VPS이면 `YES`. O.W. `NO`

- **예제**
    - 1
        - 입력
            
            ```
            6
            (())())
            (((()())()
            (()())((()))
            ((()()(()))(((())))()
            ()()()()(()()())()
            (()((())()(
            ```
            
        - 출력
            
            ```
            NO
            NO
            YES
            NO
            YES
            NO
            ```
            
    - 2
        - 입력
            
            ```
            3
            ((
            ))
            ())(()
            ```
            
        - 출력
            
            ```
            NO
            NO
            NO
            ```
